# R.E.N.D.E.R.
## Reality Engine for Non-Defined Entity Resolution

A roguelite web game featuring perspective-swapping mechanics built with React Three Fiber, Rapier Physics, and Nanostores.

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The game will open at `http://localhost:3000`

## Controls

- **Tab**: Cycle through planes (2D → ISO → FPS)
- **1/2/3**: Jump directly to a plane
- **WASD/Arrow Keys**: Move (varies by plane)
- **Space**: Jump (in 2D and FPS planes)

## Planes

- **2D (The Flatland)**: Side-scrolling platformer view
- **ISO (The Blueprint)**: Top-down tactical view
- **FPS (The Render)**: First-person immersive view
